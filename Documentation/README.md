git-absorb manual
=================

This project's documentation can be generated from [git-absorb.adoc][] as

* a man page (`git-absorb.1`) and
* an HTML document (`git-absorb.html`)

by running `make`.

Build dependencies
------------------

- [asciidoc][] (tested with version 10.2.0)
- GNU Make

[asciidoc]: https://asciidoc.org/
